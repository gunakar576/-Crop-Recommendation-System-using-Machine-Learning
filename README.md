# 🌱 Crop Recommendation System using Machine Learning

## 📌 Project Overview
This project builds a machine learning-based crop recommendation system that predicts the most suitable crop based on environmental conditions. It helps in making data-driven agricultural decisions to improve productivity and optimize resource utilization.

---

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

---

## 📊 Dataset Description
The dataset includes key environmental features influencing crop growth:
- Temperature (°C)
- Humidity (%)
- Soil pH
- Water Availability
- Season (categorical)
- Crop Label (target variable)

---

## 🤖 Model Used
- Random Forest Classifier  
- Handles non-linear relationships effectively  
- Provides feature importance for interpretability  

---

## ⚙️ Methodology
1. Data preprocessing and cleaning  
2. Encoding categorical variable (season)  
3. Splitting data into training and testing sets (80:20)  
4. Training Random Forest model  
5. Evaluating model using accuracy and classification report  
6. Analyzing feature importance  

---

## 📈 Results
- Achieved **~95–98% accuracy** on test dataset  
- Model successfully predicts suitable crops based on environmental factors  

---

## 🔍 Key Insights
- Temperature and humidity are major influencing factors  
- Soil pH plays a critical role in crop selection  
- Seasonal variations significantly impact crop suitability
